> A curated list of awesome tools, frameworks, platforms, and resources for Cloud Native technologies and development.

Cloud Native is about building and running scalable applications in modern, dynamic environments such as public, private, and hybrid clouds using technologies like containers, service meshes, microservices, immutable infrastructure, and declarative APIs.

## General Resources

- [Cloud Native Computing Foundation (CNCF)](https://www.cncf.io/) – Vendor-neutral home for open-source projects like Kubernetes, Prometheus, and more.
- [CNCF Landscape](https://landscape.cncf.io/) – Interactive map of cloud native projects and products.
- [CNCF Glossary](https://glossary.cncf.io/) – Definitions for common cloud native terms.

## Cloud Native Platforms

- [Kubernetes](https://kubernetes.io/) – Open-source container orchestration platform.
- [OpenShift](https://www.openshift.com/) – Enterprise Kubernetes platform by Red Hat.
- [Rancher](https://rancher.com/) – Kubernetes management platform for enterprises.

## Containerization

- [Docker](https://www.docker.com/) – Platform for containerizing applications.
- [Podman](https://podman.io/) – Daemonless container engine for developing, managing, and running OCI containers.
- [Buildah](https://buildah.io/) – Tool for building Open Container Initiative (OCI) container images.

## Orchestration

- [Helm](https://helm.sh/) – Kubernetes package manager.
- [Kustomize](https://kustomize.io/) – Kubernetes native configuration management.
- [Flux](https://fluxcd.io/) – GitOps Kubernetes continuous delivery.
- [ArgoCD](https://argo-cd.readthedocs.io/en/stable/) – Declarative continuous delivery tool for Kubernetes.

## Service Mesh

- [Istio](https://istio.io/) – Connect, secure, control, and observe services.
- [Linkerd](https://linkerd.io/) – Ultralight, open-source service mesh for Kubernetes.
- [Consul](https://www.consul.io/) – Service networking and discovery platform.

## CI/CD & DevOps

- [Tekton](https://tekton.dev/) – Kubernetes-native CI/CD pipelines.
- [Jenkins X](https://jenkins-x.io/) – CI/CD for modern cloud applications on Kubernetes.
- [Spinnaker](https://www.spinnaker.io/) – Multi-cloud continuous delivery platform.
- [GitLab CI/CD](https://docs.gitlab.com/ee/ci/) – Integrated CI/CD pipelines with GitLab.

## Monitoring & Observability

- [Prometheus](https://prometheus.io/) – Monitoring system and time series database.
- [Grafana](https://grafana.com/) – Visualization and dashboarding for metrics.
- [Jaeger](https://www.jaegertracing.io/) – Distributed tracing system.
- [OpenTelemetry](https://opentelemetry.io/) – Observability framework for cloud-native software.

## Security

- [Falco](https://falco.org/) – Kubernetes runtime security engine.
- [Kube-bench](https://github.com/aquasecurity/kube-bench) – Checks Kubernetes against security benchmarks.
- [OPA (Open Policy Agent)](https://www.openpolicyagent.org/) – Policy-based control for cloud native environments.

## Edge Computing

- [KubeEdge](https://kubeedge.io/) – Extends Kubernetes to edge devices.
- [OpenYurt](https://openyurt.io/) – Extends Kubernetes to edge computing scenarios.
- [SuperEdge](https://superedge.io/) – Open-source cloud-edge collaboration platform.

## Books & Learning

- [Cloud Native Patterns](https://www.manning.com/books/cloud-native-patterns) – Book by Cornelia Davis.
- [Kubernetes Up & Running](https://www.oreilly.com/library/view/kubernetes-up-and/9781491935668/) – By Kelsey Hightower, Brendan Burns, and Joe Beda.
- [The New Stack Guides](https://thenewstack.io/) – Cloud native technologies coverage.

- [Cloud Native London Meetup](https://www.meetup.com/cloud-native-london/) – Regular meetups discussing cloud native topics.
- [CNCF Slack](https://slack.cncf.io/) – Community chat for cloud native enthusiasts.
